Monefy Dash

Сервис для аналитики личного бюджета

Планируете личные или семейные финансы? Если фиксируете расходы в приложении Monefy, то импортируйте данные из приложения в сервис.

Monefy Dash визуализирует расходы и доходы за нужный период, покажет топ категорий по тратам, подсветит месяцы с самыми большими расходами и минимальными доходами. Графики помогут выявить аномалии в расходах, понять на какой категории можно сэкономить, выявить личную инфляцию, прогнозировать траты, прикинуть размер денежной подушки на случай кризиса.

Подпишитесь на канал проекта в телеграм. Там пишем о обновлениях сервиса и финансовых лайфхаках. Подписка очень поможет развитию проекта.

Импорт данных:

  1. Запустите приложение Monefy на смартфоне
  2. Перейдите в настройки
  3. Нажмите - экспорт в файл
  4. Оставьте настройки экспорта по умолчанию: UTF-8, c разделителем ';'
  5. Нажмите 'Попробовать' ниже и загрузите CSV файл из Monefy
